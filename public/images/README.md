# پوشه عکس‌ها

این پوشه برای ذخیره تمام عکس‌های پروژه استفاده می‌شود.

## ساختار:

- `menu/` - عکس‌های منو و غذاها
- `hero/` - عکس‌های صفحه اصلی
- `about/` - عکس‌های صفحه درباره ما
- `logo.png` - لوگوی رستوران (می‌توانید اینجا قرار دهید)

## نحوه استفاده:

1. عکس خود را در این پوشه قرار دهید
2. در کد از مسیر `/images/نام-فایل.jpg` استفاده کنید
3. مثال: `/images/menu/kebab-vaziri.jpg`
